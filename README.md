# RabbitWQ

A work queue built on RabbitMQ and Celluloid.

This is NOT production ready.  Released only to reserve gem name.


## Installation

Add this line to your application's Gemfile:

    gem 'rabbit-wq'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rabbit-wq
